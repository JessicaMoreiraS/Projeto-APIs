# Projeto-APIs
<p>Projetos com consumos de APIs REST</p>
<p>A página contém:</p>
<ul>
  <li>API do CEP</li>
  <li>API de moedas (Dolar, Euro e Biticoin)</li>
  <li>API do Clima</li>
  <li>Criação de API interna com busca por CPF</li>
  <li>Preenchimento de formulário com armazenamento e contagem dos dados</li>
  <li>Consumo de API com arquivo .JSON com temática de música</li>
</ul>
